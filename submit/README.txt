############# Setup #############

1. Install Python 3.7 and pip3
2. Use pip3 to install the following packages:
    - linecache
    - sys
    - numpy
    - time
    - string
    - pprint
    - itertools
    - matplotlib
    - pandas
    - sklearn
    - scipy

############## Run ##############

Use the following scripts to run the code for the corresponding section in
bwang400-analysis.pdf. The scripts for sections 1-3 accept 'wq' and 'gb' as
arguments. Pass in 'wq' to use the Wine Quality (WQ) dataset and 'gb' to use the
Generated Blobs (GB) dataset. Sections 4 and 5 don't accept arguments because
they only use the WQ dataset.
    - 1_clustering.py
    - 2_dimensionality_reduction.py
	  - 2_dimensionality_reduction_continued_1.py
	  - 2_dimensionality_reduction_continued_2.py
    - 3_all_together_now.py
    - 4_neural_network_on_dimension_reduced_data.py
    - 5_neural_network_on_cluster_reduced_data.py
